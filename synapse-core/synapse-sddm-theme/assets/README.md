Assets folder — placeholders for images, icons, fonts, and effects.

Replace these files with production-ready assets (PNG, SVG, TTF, etc.).

Structure:
- logo/: branding SVG/PNG
- wallpapers/: large background images
- icons/: action and UI icons (SVG preferred)
- fonts/: bundled fonts (optional)
- effects/: shader or effect fragments
- ui/: preview and thumbnails
